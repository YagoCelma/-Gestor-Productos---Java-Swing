# -Gestor-Productos---Java-Swing
Aplicación de escritorio para la administración de productos, ventas, clientes y empleados desarrollada con Java Swing.

Descripción

Este proyecto es una aplicación creada en Java Swing que permite gestionar diferentes áreas de un pequeño negocio: productos, ventas, clientes, empleados y el detalle de cada operación. Sirve como herramienta práctica y como ejemplo de uso de interfaces gráficas, organización por capas y conexión con base de datos.

Características principales

  -Gestión de productos (altas, bajas, modificaciones y búsqueda).
  -Registro de clientes y empleados.
  -Creación y consulta de ventas.
  -Detalle de cada venta: productos, cantidades e importes.
  -Interfaz gráfica desarrollada con Java Swing.
  -Persistencia mediante base de datos.
  -Validaciones y controles básicos.

Tecnologías utilizadas

  -Java
  -Java Swing
  -JDBC
  -Postgres


Cómo ejecutar

  -Clonar el repositorio: https://github.com/YagoCelma/-Gestor-Productos---Java-Swing.git
  -Importar el proyecto en el IDE.
  -Configurar los datos de conexión a la base de datos.
  -Ejecutar el script SQL para crear las tablas.
  -Ejecutar la clase principal del proyecto.
